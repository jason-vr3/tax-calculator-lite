# 税费及价款实时计算工具 (Tax & Price Calculator)

一款专为财务填表、商务报价以及日常计算设计的轻量级桌面工具。基于 Python 和 Tkinter 原生开发，拥有极速的响应体验和符合直觉的交互设计。

## 📝 项目简介

在处理发票、填写税务表格或进行商务报价时，我们经常需要在“价款（不含税）”、“税款”、“总价（含税）”和“税率”之间进行反复换算。本工具摒弃了繁琐的单向计算器操作，采用**“输入即所得”**的动态推导逻辑：你只需在四个参数中任意填写两个已知项，系统即可实时、精准地推导出剩余参数，并在底部展示清晰的推导算式。

## ✨ 核心特性

* **⚡ 实时双向推导**：基于输入历史队列，智能判断最后修改的两个参数并自动计算。
* **🔢 强制千分位排版**：专为财务场景优化。输入输出全局统一采用 `1,000.00` 格式，支持带逗号复制粘贴，光标移开自动排版对齐。
* **🧮 内置“双屏”计算器**：考虑到调用外部计算器可能触发杀毒软件误报，本项目纯手工重写了一个内置的简易计算器。支持运算轨迹显示（副屏记录、主屏结果），并完美支持 `Ctrl+C / Ctrl+V`。
* **🖱️ 原生交互体验**：定制的浅蓝色护眼数字小键盘，点击不抢占输入框焦点；所有输入框及输出结果均支持完整的 Windows 右键上下文菜单。
* **🛡️ 安全无毒**：纯 Python 原生 UI 实现，单文件绿色运行，不写注册表，无任何网络请求。
