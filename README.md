# 税费及价款实时计算工具 (Tax & Price Calculator)

一款专为财务填表、商务报价以及日常计算设计的轻量级桌面工具。

## 📝 项目简介

在处理发票、填写税务表格或进行商务报价时，我们经常需要在“价款（不含税）”、“税款”、“总价（含税）”和“税率”之间进行反复换算。
本工具摒弃了繁琐的单向计算器操作，采用**“输入即所得”**的动态推导逻辑。

## ✨ 核心特性

* **⚡ 实时双向推导**：基于输入队列，智能判断最后修改的两个参数并自动计算。
* **🔢 强制千分位排版**：专为财务场景优化。带逗号复制粘贴，光标移开自动排版对齐。
* **🧮 内置“双屏”计算器**：本项目纯手工重写了一个内置的简易计算器。
* **🖱️ 原生交互体验**：定制的浅蓝色护眼数字小键盘，点击不抢占输入框焦点；

## 🚀 快速使用
进入项目的 [Releases](#) 页面，下载最新的 `.exe` 可执行文件。
下载后直接双击运行即可，无需安装 Python 环境。
